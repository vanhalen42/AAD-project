DFS(Depth-First Search)

DFS (depth-first search) is a simple graph traversal algorithm. The algorithm starts at a starting node and uses the graph's edges to reach all additional nodes that are reachable from the initial node. As long as new nodes are found, a depth-first search follows a single path through the graph. It then returns to earlier nodes before moving on to other areas of the graph. The method keeps track of nodes that have been visited such that each node is only processed once.

Complexity and Data Structures

DFS has a complexity of O(m+n)O(m+n)O(m+n)﻿ where mmm﻿ is the number of edges and nnn﻿ is the number of nodes in the graph because the algorithm processes each edge and each node only once.

Where does a Stack come in here?

Notice that DFS is a "first-in, first-out(FIFO)" algorithm, in the sense that the node we reach first is the node we process first. A data structure that can help us simulate this is a stack.
